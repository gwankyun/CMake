中文翻译说明
************

依赖
=====

* CMake
* Python
* Python的Sphinx包

生成工程
========

  .. code-block:: console

    $ mkdir ../build/CMake
    $ cmake -S . -B ../build/CMake -DSPHINX_HTML=ON

编译文档
========

  .. code-block:: console

    $ cmake --build ../build/CMake --target documentation --config Debug

输出路径
========

  ``../build/CMake/Utilities/Sphinx/html``
