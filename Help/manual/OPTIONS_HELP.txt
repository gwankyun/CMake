``--help,-help,-usage,-h,-H,/?``
 打印用例信息并退出。

 用例描述了基本的命令行接口及其选项。

``--version,-version,/V [<f>]``
 显示程序名称/版本横幅并退出。

 如果指定了文件，则将版本号写入其中。|file|

``--help-full [<f>]``
 打印所有帮助手册并退出。

 所有手册皆以可读的文本格式打印。|file|

``--help-manual <man> [<f>]``
 打印一个帮助手册并退出。

 以可读的文本格式打印指定的手册。|file|

``--help-manual-list [<f>]``
 列出可用的帮助手册并退出。

 该列表包含所有可以通过使用\ ``--help-manual``\ 选项及手册名称参数获得帮助的手册。|file|

``--help-command <cmd> [<f>]``
 打印一个命令的帮助并退出。

 ``<cmd>``\ 的\ :manual:`cmake-commands(7)`\ 手册条目以可读的文本格式打印。|file|

``--help-command-list [<f>]``
 列出命令并提供帮助，然后退出。

 该列表包含所有可以通过使用\ ``--help-command``\ 选项后接命令名获得帮助的命令。|file|

``--help-commands [<f>]``
 打印cmake-commands手册并退出。

 :manual:`cmake-commands(7)`\ 手册以可读的文本格式打印。|file|

``--help-module <mod> [<f>]``
 Print help for one module and exit.

 The :manual:`cmake-modules(7)` manual entry for ``<mod>`` is printed
 in a human-readable text format.
 |file|

``--help-module-list [<f>]``
 List modules with help available and exit.

 The list contains all modules for which help may be obtained by
 using the ``--help-module`` option followed by a module name.
 |file|
